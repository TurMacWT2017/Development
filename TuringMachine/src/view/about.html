About

This is an educational Turing Machine simulator, written in Java and developed using NetBeans IDE 8.2 and Gluon SceneBuilder.

TM's language is easy for a beginner to learn, but offers the capability of solving a wide range of problems and provides good insight into the characteristics of higher-leveled programming languages.
